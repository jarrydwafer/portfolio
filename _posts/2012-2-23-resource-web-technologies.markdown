### Code schools

1. [nodeschool.io](http://nodeschool.io) - Open source workshops that teach web software skills.
2. [You Don't Know JS](https://github.com/getify/You-Dont-Know-JS) - Free book series on Javascript on Github
3. [Eggly: AngularJS Fundamentals](https://egghead.io/courses/angularjs-app-from-scratch-getting-started) - Learn to build an AngularJS application through a series of small, digestible lessons

### Technologies

1. [Craft CMS](https://craftcms.com) - Craft is a content-first CMS that aims to make life enjoyable for developers and content managers alike.
2. [Babel](https://babeljs.io) - Use next generation Javascript, today. Babel has support for the latest version of JavaScript through syntax transformers.

### Animations

1. [How SVG Line Animation Works](https://css-tricks.com/svg-line-animation-works/) - article on SVG Line Animation basics
2. [Lottie](http://airbnb.design/lottie/) - Lottie is an iOS, Android, and React Native library that renders After Effects animations in real time, allowing apps to use animations as easily as they use static images. iOS, Android, and React Native library. [great article](https://medium.com/airbnb-engineering/introducing-lottie-4ff4a0afac0e#.85rlgodw0)
3. [GreenSock](https://greensock.com) - high performance HTML5 animation for the modern web - especially this [MorphSVG](https://greensock.com/morphSVG) plugin
4. [Bounce.js](http://bouncejs.com) - Bounce.js is a tool and JS library that lets you create beautiful CSS3 powered animations.
5. [Animate.css](https://daneden.github.io/animate.css/) - CSS Animation library
6. [Vivus.js](http://maxwellito.github.io/vivus/) - a lightweight JavaScript class created by Max Wellito to animate SVG files, giving them the illusion of being drawn.
7. [Loaders.css](https://connoratherton.com/loaders) - Delightful and performance-focused pure css loading animations.
